## 🎯 概要

**将棋ウィザード**は、対局・分析・上達をサポートするモダンなウェブアプリケーションです。ポーカーのGTO Wizardにインスパイアされ、実戦と統計分析を通じて棋力向上を目指すプレイヤーを支援します。

### なぜ将棋ウィザード？

- 🎮 **どこでも対局**: インストール不要、ブラウザで完結
- ⏱️ **実戦対応**: 大会仕様の対局時計
- 📊 **成長を可視化**: 統計で自分の上達を実感
- 🎨 **美しいUI**: 滑らかなアニメーションとカスタマイズ可能なテーマ
- 💾 **プライバシー重視**: すべてローカル保存、データは自分だけのもの

## ✨ 機能

### 🎮 対局モード
- **オフライン2人対戦** 滑らかな駒移動アニメーション付き
- **合法手判定** 移動可能な場所をハイライト表示
- **王手・詰みエフェクト** 視覚的にわかりやすい演出
- **局面編集** 任意の局面から対局開始
- **棋譜保存** KIF形式での記録・読込

### ⏱️ 対局時計
- **複数の持ち時間設定**:
  - 切れ負け
  - 秒読み: 30秒、60秒など
  - フィッシャーインクリメント
  - カスタム設定
- **大きく見やすい表示** 実戦での使用に最適化
- **音声通知** 時間切れ警告
- **一時停止・再開** 機能
- **ダークモード** 低光量環境に対応

### 📊 対局統計
- **勝敗記録** 持ち時間別の成績
- **戦法レパートリー** 使用頻度分析
- **平均手数** 統計
- **時間使用パターン** の可視化
- **成長グラフ** 継続的な上達を確認
- **データエクスポート** JSON/CSV形式

### ⚙️ 設定
- **盤面テーマ**: 伝統的、モダン、ミニマルから選択
- **駒のスタイル**: 複数の書体から選択可能
- **アニメーション速度**: 好みに合わせて調整
- **効果音**: 駒音や通知のON/OFF
- **データ管理**: 対局履歴のエクスポート/インポート
